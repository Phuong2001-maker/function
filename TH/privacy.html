นโยบายความเป็นส่วนตัว

Filexo ประมวลผลทุกไฟล์ภายในเบราว์เซอร์ของคุณ เราไม่อัปโหลด ดู หรือจัดเก็บเนื้อหาของคุณ

การประมวลผลบนอุปกรณ์

ตัวแปลงดำเนินการด้วย Web API ภายในแท็บเบราว์เซอร์ ไม่มีคิวฝั่งเซิร์ฟเวอร์ ดังนั้นพลังอุปกรณ์ของคุณจึงมีขีดจำกัดเท่านั้น

ไม่มีการอัปโหลดหรือการเก็บรักษา

ไฟล์ไม่เคยเข้าถึงเซิร์ฟเวอร์ของเรา เมื่อคุณปิดแท็บ ข้อมูลการทำงานจะถูกล้างโดยอัตโนมัติ

รายละเอียดการติดต่อ

หากคุณส่งอีเมลถึงเราหรือส่งแบบฟอร์มการติดต่อ เราจะใช้ข้อมูลนั้นในการตอบกลับเท่านั้น เราไม่แบ่งปันหรือขายมัน
