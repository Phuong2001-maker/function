プライバシー ポリシー

FluxConvert はブラウザ内のすべてのファイルを処理します。コンテンツのアップロード、表示、保存は行いません。

オンデバイス処理

コンバーターはブラウザー タブ内で Web API を使用して実行されます。サーバー側のキューがないため、制限されるのはデバイスの電力だけです。

アップロードや保存はありません

ファイルがサーバーに到達することはありません。タブを閉じると、作業データは自動的に消去されます。
