Politique de confidentialité

Filexo traite chaque fichier de votre navigateur. Nous ne téléchargeons, n'affichons ni ne stockons votre contenu.

Traitement sur l'appareil

Les convertisseurs s'exécutent avec les API Web dans l'onglet du navigateur. Il n'y a pas de file d'attente côté serveur, la puissance de votre appareil est donc la seule limite.

Aucun téléchargement ni conservation

Les fichiers n'atteignent jamais nos serveurs. Une fois que vous fermez l'onglet, les données de travail sont automatiquement effacées.

Coordonnées

Si vous nous envoyez un e-mail ou soumettez le formulaire de contact, nous utilisons ces informations uniquement pour répondre. Nous ne le partageons ni ne le vendons.

Filexo

Filexo convertit les images, les PDF et les fichiers Office. fichiers directement dans votre navigateur, la seule limite est donc votre appareil. Pas de téléchargement, pas de stockage et pas d'enregistrement : juste des outils rapides qui respectent la confidentialité.
