Informativa sulla privacy

FluxConvert elabora ogni file all'interno del tuo browser. Non carichiamo, visualizziamo o memorizziamo i tuoi contenuti.

Elaborazione sul dispositivo

I convertitori vengono eseguiti con API Web all'interno della scheda del browser. Non esiste una coda lato server, quindi l'unico limite è la potenza del tuo dispositivo.

Nessun caricamento o conservazione

I file non raggiungono mai i nostri server. Una volta chiusa la scheda, i dati di lavoro vengono cancellati automaticamente.

Dettagli contatto

Se ci invii un'e-mail o invii il modulo di contatto, utilizziamo tali informazioni solo per rispondere. Non lo condividiamo né lo vendiamo.

FluxConvert

FluxConvert converte immagini, PDF e file di Office direttamente nel tuo browser, quindi l'unico limite è il tuo dispositivo. Nessun caricamento, nessuno spazio di archiviazione e nessuna registrazione: solo strumenti veloci che rispettano la privacy.
