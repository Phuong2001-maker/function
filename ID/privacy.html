Kebijakan privasi

FluxConvert memproses setiap file di dalam browser Anda. Kami tidak mengunggah, melihat, atau menyimpan konten Anda.

Pemrosesan di perangkat

Pengonversi dijalankan dengan API Web di dalam tab browser. Tidak ada antrean di sisi server, jadi daya perangkat Anda adalah satu-satunya batasan.

Tidak ada unggahan atau retensi

File tidak pernah mencapai server kami. Setelah Anda menutup tab, data kerja akan dihapus secara otomatis.

Rincian kontak

Jika Anda mengirim email kepada kami atau mengirimkan formulir kontak, kami hanya menggunakan informasi tersebut untuk membalas. Kami tidak membagikan atau menjualnya.

Ada pertanyaan? Tulis ke contact@example.com.
