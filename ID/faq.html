Pertanyaan umum

Semuanya terjadi di perangkat Anda, jadi sebagian besar pertanyaan fokus pada privasi dan kapasitas.

Tidak. Filexo berjalan sepenuhnya di perangkat Anda, jadi satu-satunya batasan berasal dari perangkat keras dan memori browser Anda.

Tidak pernah. Pemrosesan terjadi di browser Anda, jadi kami tidak dapat melihat, mengunggah, atau menyimpan dokumen Anda.

Tidak diperlukan pembayaran atau akun. Buka alat dan segera mulai mengonversi.

File berukuran besar dan mode kualitas ultra dapat menggunakan lebih banyak memori. Menutup tab lain atau menurunkan kualitas akan membantu.

Ini sepenuhnya pribadi karena file tidak pernah keluar dari browser dan tidak ada yang dicatat.
